## Description

This is a micro-fix PR to close an issue related to `HybridJudge` code that no longer exists. The bug reported in #2843 was related to a multi-line parsing issue in `HybridJudge._parse_llm_response()` found within `core/framework/graph/judge.py`.

During `Phase 2` cleanup, `core/framework/graph/judge.py` and the `HybridJudge` classes were deprecated and completely removed from the project. Therefore, the bug has been implicitly mitigated by the removal of the vulnerable code. This PR opens an empty commit referencing the issue so it can be automatically closed.
