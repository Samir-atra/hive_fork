## Description

This PR adds a new example script `core/examples/templates/iterative_pdf_processor.py` that demonstrates how to process multiple PDFs iteratively using the Hive framework. It addresses the need to process documents without exceeding context limits by defining an agent that processes PDFs one by one, accumulates outputs, and generates a trace of the execution.
