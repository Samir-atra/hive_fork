## Description

This PR implements the Regulatory Compliance Agent (GDPR RoPA + EU AI Act) template as requested in #5524. It provides a structured programmatic graph implementing the compliance workflow within `core/examples/templates/regulatory_compliance_agent.py`. The nodes included discover processing surfaces, generate Article 30 records, perform guided AI risk classification, apply human-in-the-loop review for high risk, and validate evidence packages.
