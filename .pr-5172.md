## Changes Made

- Added `starting_after` parameter to `intercom_search_conversations` and `intercom_search_contacts` tools to support pagination.
- Added `intercom_reply_conversation` tool to send a visible reply to a conversation.
- Added `intercom_update_conversation_status` tool to close, snooze, or reopen a conversation.
- Updated the credential spec for Intercom to include the new tools.
- Added corresponding unit tests in `test_intercom_tool.py`.
