## Description

This PR improves the first-time developer onboarding experience for Hive by addressing the feedback outlined in #6103. Specifically, it introduces a minimal "Hello Agent" walkthrough and adds visual diagrams representing the translation of a goal to a node graph, as well as the triangulated execution architecture (Queen, Worker, Judge).
