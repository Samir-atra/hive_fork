## Description

Implemented a native Voice Input Integration (Speech-to-Text) feature that allows users to record their voice and have the transcribed text injected directly into the chat prompt textbox. Uses `MediaRecorder` in the frontend and a new `/api/sessions/{session_id}/transcribe` backend endpoint leveraging `litellm` (Whisper API) for STT conversion.

## Changes Made
