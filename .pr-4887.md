## Description

This PR adds a `graph_version` parameter to the execution schemas (`Run`, `RunSummary`, `Checkpoint`, `CheckpointSummary`, and `SessionState`) in order to properly tie specific executions and state snapshots to an exact graph revision.

## Changes Made

- Added `graph_version: str = "1.0.0"` to `Run` schema.
- Added `graph_version: str = "1.0.0"` to `RunSummary` schema.
- Propagated `graph_version` from `Run` to `RunSummary` in `from_run`.
- Added `graph_version: str = "1.0.0"` to `Checkpoint` schema.
- Added `graph_version: str = "1.0.0"` to `CheckpointSummary` schema.
- Propagated `graph_version` from `Checkpoint` to `CheckpointSummary` in `from_checkpoint`.
- Propagated `graph_version` to the `create` method in `Checkpoint`.
- Added `graph_version: str = "1.0.0"` to `SessionState`.
