## Description

This PR adds per-environment configuration support for credentials to solve #4561. It supports dynamic configuration resolution based on the `ADEN_ENV` or `HIVE_ENV` environment variables.

## Changes Made

- Edited `core/framework/config.py` so `get_hive_config()` resolves config files such as `~/.hive/configuration.{env}.json` based on `HIVE_ENV` or `ADEN_ENV`, falling back to `configuration.json` otherwise.
- Edited `core/framework/credentials/storage.py` so `EncryptedFileStorage` checks the environment variables and creates an environment-specific base path such as `~/.hive/credentials/{env}`.
- Added tests in `core/framework/credentials/tests/test_credential_store.py` to verify environment path logic.
- Added tests in `core/tests/test_config.py` to verify configuration fallback and resolution logic.
