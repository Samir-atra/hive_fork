## Description

This PR introduces the `hive costs` CLI command, enabling users to view cost information and analyze agent spending. It adds a new `framework.costs` module containing a cost calculator that estimates LLM usage costs across multiple providers (Anthropic, OpenAI, Groq, Cerebras).
