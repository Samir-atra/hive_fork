## Description

This PR implements Semantic Behavioral Diffing & Advanced Analytics Dashboard support for Agents by logging detailed execution data for each run, specifically tracking the agent's decisions, execution path, API cost, tokens, and execution time directly into the `runs.jsonl` event log upon run completion.

## Changes Made

- Updated `RunMetrics` and `Outcome` in `core/framework/schemas/run.py` and `decision.py` to support `api_cost_usd` tracking.
- Modified `ExecutionResult` in `core/framework/graph/executor.py` to capture `api_cost_usd` and `decisions`.
- Updated `_run_execution` inside `core/framework/runtime/execution_stream.py` to gather metrics (`execution_path`, `decisions`, `execution_time_ms`, `api_tokens_used`, `api_cost_usd`) and save them directly via `_write_run_event` into `runs.jsonl`.
