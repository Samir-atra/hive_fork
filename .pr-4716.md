## Description

This PR introduces the foundation for the Visual Agent Builder (VS Code extension) by providing a generic debug protocol and standardized JSON serialization methods to stream agent execution state externally.

## Changes Made

- Add `DebugProtocol` module to `core/framework/debug_protocol.py` that serializes state tracking events to JSON-RPC structured events logic.
- Add `.to_json()` methods to `GraphSpec`, `NodeSpec`, and `EdgeSpec` inside the `core/framework/graph/` package to ensure clean extraction by the extension parser.
- Wire `event_bus` and `execution_stream` to automatically invoke `DebugProtocol` for stream started, node started, and node completed states.
- Handled potential edge cases where NodeResult could be instantiated as a dict or an object instance.
