## Description

This PR implements a robust, production-ready REST API layer using FastAPI, fulfilling the requirement for language-agnostic access to the Hive framework.

## Changes Made

- Added FastAPI, Uvicorn, and python-multipart to dependencies in `core/pyproject.toml`.
- Implemented `core/framework/api/models.py` with Pydantic models (Agent, Execution, Tool).
- Implemented `core/framework/api/middleware/` for custom error handling and request logging.
- Implemented `core/framework/api/routes/` with core endpoints for Agent CRUD, Execution flow/streaming, and Tool discovery.
- Implemented `core/framework/api/streaming/sse.py` to support real-time streaming via Server-Sent Events.
- Implemented `core/framework/api/server.py` combining all routes, health endpoints, and middleware.
- Created robust test suite in `core/framework/api/tests/test_api.py`.
