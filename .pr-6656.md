## Description

This pull request resolves an issue where the `minimax` configuration option was unavailable in the PowerShell quickstart script (`quickstart.ps1`). It aligns the options in `quickstart.ps1` with the existing setup in `quickstart.sh`, making MiniMax available as an LLM provider during initial setup on Windows.
