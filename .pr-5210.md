## Description

This PR implements Phase 1 of the formal Group-Evolving Agents (GEA) engine for adaptiveness. It establishes the pure algorithm library necessary for evolution mechanics:
1. `EvolutionArchive`: Pydantic data models (`EvolvedAgent`, `EvolutionCampaign`) and an SQLite-backed archive for persistent configuration and performance storage.
2. `Performance Vectors`: Math operations to evaluate performance (magnitude, dot product, cosine similarity/distance).
3. `Group-Level Selection`: Performance-Novelty selection to pick candidate configurations using KNN behavior clustering.
