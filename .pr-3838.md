## Description

This PR implements Phase 1 of Agent Composability by introducing the `agent_invoke` node type. This enables users to define graphs that delegate tasks to other pre-built agents directly via `agent_ref` and `input_mapping`.

## Changes Made

- Updated `NodeSpec` in `core/framework/graph/node.py` to include `agent_ref` and `input_mapping` attributes.
- Extended `VALID_NODE_TYPES` in `core/framework/graph/executor.py` to allow `"agent_invoke"` nodes.
- Added instantiation logic for `AgentInvokeNode` in `_get_node_implementation`.
- Created `AgentInvokeNode` in `core/framework/graph/agent_invoke_node.py` with scoped memory management, input mapping, and logic to run another full agent context using `EventLoopNode`.
- Created tests in `core/tests/test_agent_invoke_node.py` to ensure input and output mappings correctly persist in shared memory and that the `agent_invoke` node works as intended.
