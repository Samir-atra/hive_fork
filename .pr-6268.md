## Description

This PR adds a runnable example agent demonstrating an operational delay monitoring workflow. It resolves issue #6268 by implementing a multi-node workflow mock for:
- schedule ingestion
- delay detection
- traffic or weather context lookup
- ETA update generation
- stakeholder notification
- audit logging

This allows users to inspect the node structure, connections, and configuration directly.
