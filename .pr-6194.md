## Description

This PR introduces the `reddit_business_scanner` agent template to the `examples/templates/` directory. The agent monitors specific Reddit subreddits, filters and scores posts using an LLM to identify business opportunities, drafts customized outreach messages, and utilizes a Human-in-the-Loop (HITL) review process before logging leads (e.g. to Airtable) or sending emails.

## Changes Made

- Created `examples/templates/reddit_business_scanner/` directory with agent structure.
- Created `config.py` handling specific reddit config fields like `subreddits`, `keywords`, `score_threshold`, `outreach_tone`.
- Added tool configurations in `mcp_servers.json` for `reddit_tool`, `airtable_tool`, `gmail_tool`, `web_scrape_tool`, and `exa_search_tool`.
- Set up Node components (`fetch_node`, `score_node`, `draft_node`, `review_node`, `action_node`) to build the lead processing pipeline with hitl.
- Created graph in `agent.py` to organize goal and node execution flow.
- Setup python CLI module in `__main__.py` mapped to invoke the `reddit_business_scanner` agent via script.
- Updated `examples/README.md` to reference the new agent template.
