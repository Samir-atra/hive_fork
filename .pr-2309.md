## Description

Fixes a bug where `CredentialStore._refresh_credential` silently returns a stale token if no provider is found. The method now raises a `CredentialRefreshError` with a clear message, forcing configuration issues to be addressed immediately instead of allowing downstream applications to use expired credentials and encounter confusing `401 Unauthorized` errors.
