## Description

This PR implements an interactive Graph Visualizer for the Agent Templates, resolving the limitation of manually reading JSON files or interpreting static SVG layouts. It uses React Flow (`@xyflow/react`) and Dagre for automated top-to-bottom layouting.

## Changes Made

- Added `@xyflow/react` and `dagre` as dependencies to `core/frontend`.
- Created `GraphVisualizer` component under `core/frontend/src/components/GraphVisualizer/index.tsx`.
- Created `CustomNode` components reflecting worker names, types, and tool badges.
- Refactored `DraftGraph.tsx` to add a toggle between "Static View" (preserving all original visual features, tooltips, and layouts) and the new "Interactive View" (using React Flow).
- Logic highlighting for Start and End/Terminal nodes is provided in the new Interactive View.
