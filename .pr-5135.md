## Description

This PR introduces the **Meeting Notes & Action Item Agent**, resolving #5135. This sample agent demonstrates how to build a 4-node event loop pipeline that parses meeting transcripts, extracts structured meeting notes (executive summary, attendees, key decisions, action items, blockers, and follow-ups), allows user review, and optionally delivers the notes to a Slack channel via a FastMCP tool integration.
