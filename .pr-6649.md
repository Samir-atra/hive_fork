## Description

Fixes the conversation judge silent bypassing of quality gates when the LLM provider fails (e.g., due to rate limits or malformed responses). It implements a retry-then-degrade mechanism. On the first failure, it returns `RETRY` (with `confidence=0.0`) to give the LLM another chance. On the second consecutive failure, it degrades to `ACCEPT` (with `confidence=0.0`) with an explicit system feedback message.
