## Description

This PR implements the requested PagerDuty integration MVP functions (`trigger_incident`, `get_on_call`, and `resolve_incident`) to enable Hive agents to directly interact with PagerDuty's dispatch system. Resolves #2891.

## Changes Made

- Added `trigger_incident` to `pagerduty_tool.py` allowing creation of new incidents.
- Added `get_on_call` to `pagerduty_tool.py` to retrieve on-call users given a schedule ID.
- Added `resolve_incident` to `pagerduty_tool.py` to resolve existing incidents.
- Updated unit tests in `test_pagerduty_tool.py` to cover new features correctly.
