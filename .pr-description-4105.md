## Summary

This PR implements the **Closed-Loop Agent Evolution** system as described in issue #4105. This is a three-phase system that enables Hive agents to learn from every execution and improve over time.

### Phase 1: Deterministic Execution Tracing

- **TraceCapture**: Middleware to capture full LLM/tool I/O during execution
- **ExecutionTrace**: Complete trace model with all interactions (LLM, tools, node boundaries)
- **TraceStore**: Persistent storage for execution traces
- **ReplayEngine**: Re-run past executions with deterministic stubs for validation

### Phase 2: Episodic Memory with Vector Retrieval

- **Episode**: Structured context + action + outcome tuple
- **EpisodeWriter**: Captures episodes from RuntimeLogger automatically
- **EpisodeRetriever**: Retrieves relevant past experiences for LLM context injection
- **EpisodicMemoryStore**: Persistent storage with vector indexing
- **VectorBackend**: Pluggable backend interface (ChromaDB default, FAISS, InMemory alternatives)

### Phase 3: Self-Evolving Optimization

- **AgentConfiguration**: Configurable agent aspects (prompts, thresholds, rules)
- **FitnessFunction**: Evaluates configuration fitness using multiple signals
- **PopulationManager**: Manages configuration variants with tournament selection
- **MutationOperator**: LLM-guided configuration mutations preserving semantic validity
- **EvolutionShadowRunner**: Validates configurations against historical traces
- **EvolutionPipeline**: Orchestrates the full evolution cycle with HITL approval gates

## Usage Example

```python
from framework.tracing import TraceCapture, TraceStore
from framework.memory import EpisodicMemoryStore, EpisodeWriter
from framework.evolution import EvolutionPipeline, EvolutionConfig

# Initialize components
trace_store = TraceStore(base_path=Path("traces"))
memory_store = EpisodicMemoryStore(base_path=Path("episodes"))

# Run evolution pipeline
pipeline = EvolutionPipeline(
    agent_id="my_agent",
    trace_store=trace_store,
    memory_store=memory_store,
    llm_provider=llm,
    config=EvolutionConfig(
        population_size=10,
        max_generations=50,
        require_hitl_approval=True,
    ),
)

result = await pipeline.run()
if result.deployed:
    print(f"Deployed config {result.deployed_config_id}")
```
