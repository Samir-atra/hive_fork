## Description

This PR implements Phase 2 of the Google Analytics integration as requested in #4244. It adds filtering and ordering to `ga_run_report`, and introduces `ga_compare_date_ranges` and `ga_list_properties`.

## Changes Made

- Updated `ga_run_report` to support `dimension_filter`, `metric_filter`, and `order_bys` parameters.
- Added new tool `ga_compare_date_ranges` to compare metrics across two date ranges.
- Added new tool `ga_list_properties` to list properties accessible by the service account using the Admin API.
- Updated documentation in `google_analytics_tool/README.md`.
- Added related unit tests for the new parameters and tools.
