## Description

Implement the v1 baseline for the Unified Agent Evaluation Framework as discussed in issue 5372. This introduces a formal evaluation schema and metrics tracking for every agent execution, operating post-run.

## Changes Made

- Created `EvaluationResult` schema and `FailureTaxonomy` in `core/framework/schemas/evaluation.py`.
- Integrated `EvaluationResult` into `Run`, `RunSummary`, and `SessionState` schemas.
- Hooked up `EvaluationResult` creation at `end_run` for both `Runtime` (`core.py`) and `StreamRuntime` (`stream_runtime.py`).
- Added robust tests in `core/framework/runtime/tests/test_evaluation.py` tracking logic to test evaluation properties.
