## Description

Add Research & Summarization Example Agent (#4988)
Resolves #4988

This PR adds a multi-step `research_summarization_agent.py` example agent to `core/examples/templates`. The agent simulates web research, content extraction, and summarization using pure Python implementations of the `NodeProtocol` interface, serving as an educational example of how to orchestrate a basic workflow using the core framework.
