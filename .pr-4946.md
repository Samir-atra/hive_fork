## Description

This PR introduces an optional circuit breaker for LLM and MCP calls. It implements a `CircuitBreaker` utility class that tracks failures and opens after a configurable threshold, immediately failing subsequent requests and preventing the framework from repeatedly hitting degraded dependencies.
