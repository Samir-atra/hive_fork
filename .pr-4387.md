## Description

Refactors the `_handle_response` method in the Apollo tool to properly separate HTTP protocol errors from Apollo-specific logic and returns a structured response format.

## Changes Made

- Refactored `_handle_response` in `apollo_tool.py` to return a structured dictionary containing `success`, `error`, `status_code`, `details`, and `retryable`.
- Updated all Apollo tool methods (`enrich_person`, `enrich_company`, `search_people`, `search_companies`, etc.) to use the structured format and correctly parse response payload under `result["data"]`.
- Updated tests in `test_apollo_tool.py` to assert against the new structured HTTP error responses.
