## Description

This PR enables parallel test execution using `pytest-xdist` with the `-n auto` flag. By adding this flag to the CI workflows (`ci.yml`, `release.yml`) and `Makefile`, test execution duration will be reduced.

## Changes Made

- Updated `Makefile` to add `-n auto` to pytest commands.
- Updated `.github/workflows/ci.yml` and `.github/workflows/release.yml` to include `-n auto` for parallel test execution.
- Added `pytest-xdist>=3.0` to the `dev` dependency group in `tools/pyproject.toml`.
- Configured `asyncio_mode = "auto"` in `core/pyproject.toml` to ensure asynchronous tests are compatible with parallel execution.
