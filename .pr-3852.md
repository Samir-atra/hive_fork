## Description

This PR introduces a new `hive visualize` command that enables interactive web-based debugging of agent graphs. It allows for statically viewing an agent's graph, as well as live execution visualization utilizing WebSockets to stream events directly to a local D3.js powered UI.

## Changes Made

- Added `viz = ["websockets>=12.0"]` optional dependency to `pyproject.toml`
- Added `VisualizerServer` inside `core/framework/visualizer/server.py` to handle broadcasting node events to a WebSocket.
- Added a static, 0-build frontend inside `core/framework/visualizer/index.html` using D3.js and Dagre-D3.
- Added `visualize.py` CLI command entry and registered it in `core/framework/cli.py`.
- Hooked `GraphExecutor` `execute` methods to emit start/complete events to `self.visualizer` if provided.
- Tested the newly added components for functionality and dependency failures.
