## Description

This PR introduces MCP tool context usage tracking and analytics for Hive, fulfilling #3553. The implementation adds a static token measurement script and a runtime `ContextUsageTracker` that optionally estimates dynamic tokens during execution. A new MCP tool `get_context_usage` is exposed to access this summary during a session.

## Changes Made

- Added `measure_context_usage.py` in `tools/scripts/` to generate `TOOL_CONTEXT_REPORT.md` and `tool_context_usage.json` with a heuristic of 4 characters ≈ 1 token.
- Created `ContextUsageTracker` in `aden_tools.monitoring.context_tracker` to track static registration tokens and dynamic execution tokens.
- Modified `register_all_tools` in `tools/src/aden_tools/tools/__init__.py` to integrate static token tracking and a monkey-patch execution wrapper for dynamic tool tracking without depending on missing middleware.
- Created `get_context_usage` MCP tool in `aden_tools.tools.context_tracking_tool` to allow agents and users to view resource utilization.
