## Description

Add document ingestion MCP tools (`pdf_extract_text`, `docx_extract`, `xlsx_read`) to extract text and data from common formats safely with strict size caps and pagination, matching the feature request #4208.

## Changes Made

- Add `pdf_extract_text` to extract pages with truncation limits.
- Add `docx_extract` to extract paragraphs with pagination constraints.
- Add `xlsx_read` to read specified sheets and apply row constraints.
- Updated `tools/src/aden_tools/tools/__init__.py` to register the new tools.
- Add tests for `pdf_extract_text`, `docx_extract`, and `xlsx_read` under `tools/tests/tools/test_document_ingestion.py`.
