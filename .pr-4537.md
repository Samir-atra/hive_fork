## Description

This PR resolves #4537 by implementing the Goal-Driven Inbound Lead Sentinel agent template. This newly added agent addresses "lead leakage" by automatically managing inbound demo requests. It qualifies requests by enriching data via an Apollo.io mock and running a dynamic ICP scoring logic that calculates priorities based on revenue, industry, and size constraints. The agent incorporates a reliable circuit breaker and safely routes the appropriate high-scoring prospects to Salesforce as opportunities.
