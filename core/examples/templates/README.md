# Examples & Templates

This directory contains various example agents and templates built using the Hive framework.
These are designed to act as practical starter examples for developers exploring Hive for real-world use cases.

## Included Examples

- **Research Summarization Agent**: A multi-step workflow that simulates taking a topic as input, finding relevant sources, extracting information, and generating a structured summary.
  - Usage: `uv run python core/examples/templates/research_summarization_agent.py`
