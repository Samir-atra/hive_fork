## Description

This PR implements a comprehensive validation framework for checking the workflow Intermediate Representation (IR) graphs produced by the NL compiler. It ensures structural errors like DAG cycles, unreachable tasks, missing dependencies, and schema validity issues are caught before execution. It also includes property-based testing (via the `hypothesis` framework) to strictly ensure invariants hold over generated graphs.
