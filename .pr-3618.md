## Description

This PR introduces a commercial-ready agent template called `commercial_crm_agent`. This template acts as a "Revenue/Support Agent" that handles queries connecting a CRM (like HubSpot) directly to messaging channels (like Slack) without the need to write standard Python integration logic.
